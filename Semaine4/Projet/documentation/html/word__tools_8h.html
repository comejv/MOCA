mot_data_t *  w2 
)

Fonction de comparaison de deux mots.

Parameters
w1la structure du premier mot
w2la structure du second mot
Returns
Renvoie 1 si w1 est vide, si le caractère de position i dans w1 a un code ASCII strictement supérieur au caractère
de même position dans w2, ou si w1 est strictement plus long que w2
Renvoie -1 si w2 est vide, si le caractère de position i dans w1 a un code ASCII strictement inférieur au caractère
de même position dans w2, ou si w1 est strictement moins long que w2
Renvoie 0 si w1 et w2 décrivent le même mot

Cette fonction compare deux mots à partir de leur représentation textuelle (chaîne de caractères).
