Fonction de création d'une structure de mot à partir de données sur son contenu et sa position dans un texte.

Parameters
dictionarydictionnaire qui reçoit le mot
wordchaîne de caractère associée au contenu du mot
lineentier associé à la ligne où se trouve le mot dans le texte
colonneentier associé à la colonne où se trouve le mot dans le texte
Note
Pas de valeur de retour

Cette fonction crée une structure mot_t à partir d'une chaîne de caractères (contenu du mot) et de deux entiers (numéros de ligne et de colonne où se trouve le mot), puis l'insère dans un dictionnaire dico, en créant ce dictionnaire si le pointeur est nul ou en appelant insertDico sinon.
